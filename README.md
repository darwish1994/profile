# Ahmed Darwish - Personal Website

A modern, responsive personal portfolio website for Ahmed Darwish, showcasing his experience as an Android Developer.

## Features

- Responsive design that works on mobile, tablet, and desktop devices
- Clean, modern UI with smooth animations
- Sections for About, Experience, Skills, Education, and Contact
- Contact form for potential employers or clients to reach out
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid layouts)
- JavaScript (ES6+)
- Font Awesome icons

## Getting Started

### Prerequisites

No special prerequisites are needed to run this website. Any modern web browser will work.

### Installation

1. Clone or download this repository
2. Open the `index.html` file in your web browser

### Deployment

This website can be deployed to any web hosting service. Simply upload all files to your hosting provider.

Popular hosting options include:
- GitHub Pages (free)
- Netlify (free tier available)
- Vercel (free tier available)
- Any traditional web hosting service

## Customization

### Updating Content

To update the content of the website:

1. Edit the `index.html` file to change text content
2. Modify the `styles.css` file to change the appearance
3. Update the `script.js` file to modify functionality

### Adding Projects

You may want to add a Projects section to showcase your Android applications. To do this:

1. Add a new section in the `index.html` file
2. Style it appropriately in the `styles.css` file
3. Update the navigation menu to include the new section
